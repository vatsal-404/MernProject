# 🛒 MERN E-Commerce Platform

A feature-rich, full-stack e-commerce solution built with the **MERN stack**. This platform offers a seamless shopping experience with secure authentication, real-time stock management, and an intuitive admin interface.

## ✨ Features

### **User Experience**
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
* **Google Auth:** Secure and quick login using **Firebase Authentication**.
* **Shopping Workflow:** Seamless "Add to Cart" functionality and structured checkout process.
* **AI Recommendations:** Smart product suggestions to enhance user engagement.

### **Management & Admin**
* **Admin Dashboard:** Centralized control for managing **Products, Users, and Orders**.
* **Stock Management:** Real-time updates on product availability.
* **Order Tracking:** Users can view and manage their order history and bookings.

### **Technical Excellence**
* **RESTful API:** Robust backend architecture for efficient data handling.
* **Scalable DB:** Flexible document-based storage with MongoDB.
